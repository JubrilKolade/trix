# Mobile Support

<span className="inline-flex items-center gap-2 px-2 py-1 rounded-full text-xs font-bold bg-primary/10 text-primary border border-primary/20">Official Support</span>

Trix provides first-class support for React Native and Flutter, allowing you to bootstrap mobile applications with the same ease as web and backend projects.

Please choose a platform to get started:

- [React Native](/docs/mobile/react-native)
